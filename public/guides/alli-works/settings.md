# 설정 가이드

Alli Works의 전반적인 설정을 관리하는 방법을 설명합니다.

---

## 1. 일반 설정

**경로**: 설정 > 일반

| 항목 | 설명 |
|------|------|
| **프로젝트 이름** | 프로젝트 표시 이름 |
| **기본 언어** | 프로젝트 기본 언어 설정 |
| **타임존** | 시간대 설정 |

---

## 2. 멤버 및 그룹

### 멤버 관리
**경로**: 설정 > 멤버 및 그룹 > 멤버

- 멤버 초대: 이메일로 초대 발송
- 역할 지정: 관리자 / 편집자 / 뷰어
- 멤버 비활성화/삭제

### 그룹 관리
**경로**: 설정 > 멤버 및 그룹 > 그룹

- 부서/팀 단위 그룹 생성
- 그룹별 문서 접근 권한 설정
- 앱 접근 권한의 단위로 활용

---

## 3. 보안

### 기본 보안 설정
**경로**: 설정 > 보안

- 세션 타임아웃 설정
- IP 제한
- 2단계 인증

### 가드레일 기능
**경로**: 설정 > 보안 > 가드레일

AI 응답에 민감한 정보가 포함되지 않도록 필터링 규칙을 설정합니다.

| 가드레일 유형 | 설명 | 예시 |
|--------------|------|------|
| **개인정보 필터** | 이름, 주민번호 등 마스킹 | 홍길동 → *** |
| **금융정보 필터** | 계좌번호, 카드번호 마스킹 | |
| **미공개 정보 필터** | 미공개 중요정보 차단 | |
| **커스텀 규칙** | 특정 키워드/패턴 차단 | |

#### 설정 방법
1. 가드레일 규칙 추가
2. 필터 유형 선택 (차단 / 마스킹 / 경고)
3. 대상 패턴 정의 (정규식 또는 키워드)
4. 적용 범위 설정 (전체 / 특정 앱)

---

## 4. 변수 관리

**경로**: 설정 > 변수

### 변수 유형

| 유형 | 설명 | 예시 |
|------|------|------|
| **문자열 변수** | 텍스트 값 저장 | @TEXT, @ANSWER |
| **목록 변수** | 복수 값 저장 | @SELECTED_ITEMS |
| **파일 변수** | 파일 참조 저장 | @UPLOADED_FILE |

### 시스템 변수
Alli가 자동으로 제공하는 변수:
- `@USER_NAME`: 현재 사용자 이름
- `@USER_EMAIL`: 현재 사용자 이메일
- `@CURRENT_DATE`: 현재 날짜

### 변수 활용
- 답변 생성 노드: 답변을 변수에 저장
- 조건 추가 노드: 변수값으로 분기
- LLM 노드: 프롬프트에 변수 삽입 ({@변수명})
- 메시지 보내기: 메시지에 변수값 표시

---

## 5. Alli Works 설정

**경로**: 설정 > Alli Works 설정

### 브랜딩
| 항목 | 설명 |
|------|------|
| **로고** | Works 화면에 표시할 로고 이미지 |
| **테마 색상** | UI 주요 색상 커스터마이징 |
| **첫 화면 안내 문구** | 홈 화면에 표시할 환영 메시지 |

### 커스텀 도메인
- 자체 도메인으로 Alli Works 접속 가능
- SSL 인증서 자동 적용

### 카테고리 관리
- 앱을 분류할 카테고리 생성/관리
- 사용자 홈 화면에 카테고리별로 앱 표시

---

## 6. 인증 설정

### 인증 방식

| 방식 | 설명 |
|------|------|
| **통합인증 (ID/PW)** | 기본 아이디/비밀번호 로그인 |
| **SAML SSO (Google)** | Google Workspace SSO |
| **SAML SSO (Entra)** | Microsoft Entra ID SSO |
| **SAML SSO (Okta)** | Okta SSO |

### SSO 설정 시 유저 속성 매핑
- 이름, 이메일, 부서 등 속성을 자동 매핑
- 신규 사용자 자동 생성 설정 가능

---

## 7. 프롬프트 관리

**경로**: 설정 > 프롬프트 관리 (또는 대시보드 > 프롬프트 관리)

### 앱 프롬프트
- 특정 앱에 적용되는 시스템 프롬프트
- 앱의 역할, 응답 스타일, 제약사항 정의

### 프로젝트 프롬프트
- 프로젝트 전체에 적용되는 공통 프롬프트
- 모든 앱에 공통으로 적용할 가이드라인

---

## 8. 결제 및 크레딧

**경로**: 설정 > 결제

- 현재 구독 플랜 확인
- 크레딧 사용량 모니터링
- 크레딧 제한 설정 (멤버별/앱별)
