## Alli Works 주요 화면 구성

### 1. 메인 화면
- **왼쪽 사이드바**: 프로젝트 목록, 새 대화 시작, 대화 히스토리, 문서함, 관리 메뉴
- **중앙 대화 영역**: 질문 입력 및 AI 답변 표시
- **우측 출처 패널**: RAG 기반 답변의 출처 문서 확인

### 2. 홈 화면
- **안내 문구**: 프로젝트별 맞춤 안내 메시지
- **추천 질문**: 자주 묻는 질문/대화 주제 추천
- **앱 카테고리**: 추천/즐겨찾기/전체 앱 목록

### 3. 문서함
- **공용 문서함**: 관리자가 업로드한 부서별 공유 문서
- **개인 문서함**: 개인 업무용 문서 관리 (본인만 접근 가능)
- **검색 범위 설정**: 대화 시 참조할 문서를 자유롭게 선택

### 4. 대화 기능
- **출처 정보**: 답변 근거 문서 확인 및 다운로드
- **썸네일 기능**: 출처 문서 전체 페이지 미리보기
- **피드백 시스템**: 👍/👎 피드백으로 답변 품질 지속 개선
