## 파트 3. 공용 AI vs 사내 승인 도구 사용 경로

```
입력하려는 데이터가 있을 때:

Q1. 이 데이터가 공개 정보인가?
  → NO: Q2로
  → YES: 공용 AI (ChatGPT 등) 사용 가능 🟢

Q2. 이 데이터에 고객 실명·계좌·MNPI가 포함되어 있는가?
  → YES: 절대 입력 금지 🔴 (비식별화해도 원칙 위반)
  → NO: Q3로

Q3. 사내 비공개 정보이지만 MNPI·개인정보는 아닌가?
  → YES: 사내 승인 AI 도구로만 🟡
  → NO: 공용 AI 사용 가능 🟢 (재확인 후)
```

## 파트 4. AI 출력 검증 3단계 (라미네이팅 버전용)

```
────────────────────────────────────────
  AI 출력물 사용 전 반드시 이 3단계를 거치세요
────────────────────────────────────────

STEP 1. 수치·사실 확인
  [ ] 숫자, 날짜, 법령 조항이 있으면 원출처로 대조
  [ ] 출처: DART, Bloomberg, 한은, 금감원 공식 사이트

STEP 2. 컴플 자가 체크
  [ ] 고객 개인정보 섞여 있지 않은가?
  [ ] MNPI / 미공개 딜 정보 포함되지 않았는가?
  [ ] AI가 아닌 내가 작성한 것처럼 책임질 수 있는가?

STEP 3. 상급자 리뷰 전 최종 감수
  [ ] 외부 발송·내부 보고 전 최소 1인 검토
  [ ] 중요 수치·법령은 담당자 직접 확인

────────────────────────────────────────
```

## 파트 6. 자주 묻는 질문 (FAQ)

**Q. ChatGPT 무료 버전과 유료 버전의 보안 수준이 다른가요?**
A. 유료(Plus, Team) 플랜은 기본적으로 학습 데이터 사용을 끄는 옵션이 있지만, 하나증권의 보안 정책은 도구의 유·무료 여부와 무관하게 적용됩니다. MNPI·고객정보는 어느 플랜에서도 입력 금지입니다.

**Q. 비식별화하면 고객 정보를 입력해도 되나요?**
A. 완전 비식별화(실명, 계좌, 연락처 제거 + 집계 통계로 변환)된 데이터는 노랑(🟡) 판단 대상입니다. 그래도 사내 승인 도구 우선 사용을 권장하며, 공용 AI 사용 전 컴플 팀 확인을 권합니다.

**Q. AI가 생성한 문서에 제 이름을 붙여 제출해도 되나요?**
A. AI는 보조 도구입니다. 최종 문서의 책임은 작성자인 여러분에게 있습니다. AI 초안을 검토·수정·검증 없이 그대로 제출하는 것은 권장하지 않으며, 특히 수치·법령 인용이 포함된 경우 반드시 검증 3단계를 거쳐야 합니다.

**Q. 사내 승인 AI 도구가 무엇인지 어디서 확인하나요?**
A. IT팀 또는 컴플라이언스팀에 문의하거나 사내 인트라넷 공지를 확인하세요. 승인되지 않은 AI SaaS 도구를 사내 데이터 처리에 사용하는 것은 정보보안 정책 위반입니다.

**Q. 외부 발간된 증권사 리서치 보고서를 ChatGPT에 붙여넣어도 되나요?**
A. 타사 리서치 보고서가 외부에 공식 배포된 경우 공개 정보로 분류되어 초록(🟢) 적용 가능합니다. 단, 자사 미발간 보고서는 🔴입니다.
