## M2 후반 — 보안 심화·퀴즈·검증 루틴 (2교시, 50분)

### 학습 목표
- 3색 신호등(빨강/노랑/초록) 기준을 외우고 즉시 적용할 수 있다
- MNPI, 고객 개인정보, 발간 전 정보 등 5가지 금지 카테고리를 설명할 수 있다
- AI 출력이 "사실"이 아닌 "초안"임을 인지하고 검증 3단계를 수행할 수 있다
- 보안 퀴즈 20문항에서 90% 이상 정답을 맞힐 수 있다

### 구성

| 시간 | 내용 | 방식 |
|------|------|------|
| 8분 | 5개 금지 카테고리 ④⑤ 설명 (AI 환각·사내 비공개 자료) + 사례 실증 | 강의 |
| 5분 | "공용 AI vs 사내 승인 도구" 결정 플로차트 | 강의 |
| 15분 | 신호등 퀴즈 20문항 — 개인 풀기 → 정답 공개 | 퀴즈 |
| 17분 | 퀴즈 해설 (빨강 케이스 중심, 대안 제시) | 해설 |
| 5분 | 검증 3단계 루틴 암기 + 체크리스트 배포 | 강의 |

### 5개 금지 카테고리 상세

#### 카테고리 1: MNPI (미공개중요정보)
- **정의**: 공시 전 실적, M&A 협상 내용, 내부 경영 계획, 증자 계획 등
- **왜 위험한가**: 자본시장법 제174조 위반, 내부자거래 혐의, AI 서버에 전송 = 유출
- **케이스**: "우리 고객사 다음 주 공시 예정인 영업이익 수치로 보고서 초안 써줘" → 🔴
- **대안**: 공시된 이후 수치로만 작업

#### 카테고리 2: 고객 개인정보·금융정보
- **정의**: 고객 실명, 계좌번호, 잔고, 거래내역, 연락처, 포트폴리오 구성
- **왜 위험한가**: 개인정보보호법·신용정보법·금융소비자보호법 위반, 과징금·형사처벌
- **케이스**: "이 고객(홍길동, 계좌 123456)에게 맞는 포트폴리오 추천해줘" → 🔴
- **대안**: 가상의 고객 프로필("50대, 자산 5억, 중위험 성향")로 대체해 작업

#### 카테고리 3: 발간 전 리서치·딜 정보
- **정의**: 미발간 분석 보고서, 목표주가, 딜 코드명, IM 원문, 실사 자료
- **왜 위험한가**: 정보 유출·이해상충, 조사분석자료 관련 규정 위반
- **케이스**: "우리 팀 곧 나올 OO기업 리포트 초안 요약해줘" → 🔴
- **대안**: 이미 배포·발간된 리포트, 공시 자료만 사용

#### 카테고리 4: AI 출력 무검증 사용
- **정의**: 환각 위험 — AI가 자신 있게 틀린 수치·법령·날짜를 생성
- **왜 위험한가**: 고객 손실, 보고서 오류, 임원 보고 실수
- **케이스**: AI가 "A기업 2024 영업이익 1.2조"라고 했는데 실제는 0.8조
- **대안**: 수치는 반드시 원출처(금감원 DART, Bloomberg 등)로 대조 검증

#### 카테고리 5: 사내 비공개 자료 공용 AI 입력
- **정의**: 사내 매뉴얼, 계약서, 인사 자료, 감사 자료, 내부 규정 전문
- **왜 위험한가**: 영업비밀 유출, 계약 위반, 로그에 남아 컴플 모니터링 대상
- **케이스**: "우리 회사 내부 감사 보고서 요약해줘 [파일 첨부]" → 공용 AI에서 🔴
- **대안**: 사내 승인 도구(노랑) 또는 요약 없이 직접 읽기

### 3색 신호등 퀴즈 구성 원칙
- 총 20문항, 각 케이스를 읽고 빨강/노랑/초록 중 선택
- 문항 배분: 빨강 10개, 노랑 5개, 초록 5개
- 정답 공개 후 빨강 케이스는 반드시 "대안" 설명 병행
- 퀴즈 결과지는 수강생 보관 + 강사 집계

### 검증 3단계 루틴
```
Step 1. 수치·사실 확인
  → AI 출력에 수치, 날짜, 법령 조항이 있으면 원출처로 100% 대조

Step 2. 컴플 자가 체크
  → 이 출력물에 고객 개인정보 / MNPI / 미공개 딜 정보가 섞여 있지 않은가?

Step 3. 상급자 리뷰 전 최종 감수
  → 외부 발송·보고 전 최소 1인 이상 검토
```

## M3. AI 동작 원리 & 프롬프팅 기본기 (3교시, 50분)

### 학습 목표
- LLM이 어떻게 작동하는지 비개발자도 이해할 수 있는 수준으로 설명할 수 있다
- 좋은 프롬프트의 4요소(역할·맥락·지시·형식)를 알고 업무 프롬프트에 적용할 수 있다
- Before/After 프롬프트를 비교하고 개선 포인트를 지적할 수 있다

### 구성

| 시간 | 내용 | 방식 |
|------|------|------|
| 10분 | LLM 작동 원리 — "다음 단어를 예측하는 기계" 비유로 설명 | 강의 |
| 5분 | AI의 3가지 한계: 환각, 지식 컷오프, 맥락 창 | 강의 |
| 15분 | 프롬프팅 4요소: 역할(Persona) / 맥락(Context) / 지시(Task) / 형식(Format) | 강의 + 예시 |
| 13분 | Before/After 프롬프트 비교 실습 (3개 사례) | 참여형 |
| 7분 | 프롬프트 개선 팁 5가지 + 흔한 실수 3가지 + 3교시 정리 | 강의 |

### 프롬프팅 4요소 예시 (증권업무 맥락)

```
[역할(Persona)]
"당신은 국내 대형 증권사 WM 부서의 시니어 어드바이저입니다."

[맥락(Context)]
"한국은행이 오늘 기준금리를 0.25% 인하 발표했습니다.
다음은 관련 보도자료 요약입니다: [요약 붙여넣기]"

[지시(Task)]
"위 내용을 바탕으로 채권형 펀드를 보유한 고객에게 보낼
주의사항 안내 메일 초안을 작성해 주세요."

[형식(Format)]
"분량 200자 이내, 전문 용어 최소화, 불릿 포인트 없이 자연스러운 문장체로."
```

### Before/After 비교 사례

| # | Before (나쁜 프롬프트) | After (좋은 프롬프트) |
|---|----------------------|----------------------|
| 1 | "채권 뭐가 좋아?" | "2026년 4월 기준 금리 인하 사이클에서 국내 채권형 ETF 3종을 장단점 비교해줘. Bloomberg 공개 데이터 기준, 표 형식으로." |
| 2 | "보고서 써줘" | "첨부한 공시 PDF(삼성전자 2025 사업보고서 발췌)를 읽고 매출·영업이익·부채비율을 표로 정리해줘. 숫자는 원 단위로." |
| 3 | "번역해줘" | "다음 영문 리서치 제목과 요약 3단락을 한국어로 번역해줘. 금융 용어는 국내 증권업계 표준 용어로 맞춰줘." |

## M5. 핸즈온 — 내 업무 1개를 AI 워크플로로 (5교시 전반, 35분)

### 학습 목표
- 실제 또는 모의 업무 시나리오에서 안전한 AI 워크플로를 직접 설계하고 실행할 수 있다
- 프롬프트 작성 → AI 실행 → 검증 3단계 → 컴플 체크의 전체 사이클을 경험한다

### 트랙 선택

**트랙 A — 노트북 실습 가능 시**

| 시간 | 내용 |
|------|------|
| 5분 | 가상 업무 시나리오 안내 (부서별 모의 데이터 배포) |
| 15분 | 개인 실습: 시나리오 → 프롬프트 설계 → AI 실행 → 검증 |
| 10분 | 조별 결과 발표 (2~3분/조): 프롬프트 + 결과 + "무엇을 검증했나" |
| 5분 | 강사 피드백 + 베스트 프롬프트 선정 |

**트랙 B — 강의장 실습 불가 시**

| 시간 | 내용 |
|------|------|
| 8분 | 강사 라이브 시연: 잘된 프롬프트 vs 나쁜 프롬프트 화면 비교 |
| 15분 | 조별 종이 워크시트: "내 부서 시나리오에 맞는 프롬프트 작성" |
| 8분 | 조별 발표 + 강사 코멘트 |
| 4분 | 베스트 프롬프트 공유 + 배포 |

### 가상 업무 시나리오 (트랙 A 기준)

| 부서 | 시나리오 | 사용 데이터 |

## M6. 마무리·체크리스트·Q&A (5교시 후반, 15분)

### 학습 목표
- 오늘 배운 핵심 3가지를 자신의 말로 요약할 수 있다
- 사무실 책상에 붙여둘 1페이지 체크리스트를 손에 쥔다

### 구성

| 시간 | 내용 |
|------|------|
| 5분 | 오늘의 핵심 3가지 복습 슬라이드 |
| 3분 | 보안 체크리스트 1페이지 배포 + 설명 |
| 5분 | Q&A (사전 익명 질문 + 현장 질문) |
| 2분 | 마무리: "여러분은 오늘부터 AI를 안전하게 쓸 면허를 받았습니다" |

### 오늘의 핵심 3가지 (수강생 기억 앵커)
1. **🔴 빨강은 무조건 멈춰** — MNPI·고객정보·발간 전 자료는 어떤 AI에도 입력 금지
2. **📝 초안은 초안이다** — AI 출력은 반드시 검증 3단계 후 사용
3. **🚀 초록 영역을 적극 활용해** — 공개 정보 처리·번역·초안 작성은 지금 당장 써라
