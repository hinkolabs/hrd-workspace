## 부록: Q&A 예상 질문 & 답변 준비

**Q. ChatGPT 유료 플랜(Plus, Team)은 보안 수준이 달라서 써도 되나요?**

A. 유료 플랜은 학습 데이터 사용을 끄는 옵션이 있지만, 하나증권 보안 정책은 도구의 유·무료 여부와 무관합니다. MNPI·고객 정보는 어떤 플랜에서도 입력 불가입니다. 사내 정책이 외부 서비스 약관보다 우선합니다.

**Q. 비식별화 완벽하게 하면 고객 정보 넣어도 되나요?**

A. 완전 비식별화(실명·계좌·연락처 제거 + 집계 통계로 변환)된 데이터는 노랑(🟡) 판단 대상입니다. 공용 AI에서는 여전히 안 되고, 사내 승인 도구를 우선 사용하세요. 애매하면 컴플 팀에 먼저 확인하세요.

**Q. 사내 승인 AI 도구가 뭔가요? 없는데요?**

A. 현재 사내 공식 AI 도구 현황은 IT팀 또는 컴플 팀에 확인하세요. 아직 없거나 모른다면, 지금은 공개 정보에 한해서만 공용 AI를 쓰시고, 내부 데이터 처리가 필요한 경우는 IT팀에 공식 요청하세요.

**Q. AI가 만든 문서에 제 이름을 붙여도 되나요? 저작권 문제는요?**

A. 최종 문서의 책임과 저작권은 여러분에게 있습니다. AI는 보조 도구예요. AI 초안을 검토·수정·검증 없이 그대로 제출하는 건 권장하지 않습니다. 수치·법령이 포함된 경우 반드시 검증 3단계를 거치세요.

**Q. 핸즈온 때 쓴 가상 데이터를 실제 업무에 그대로 써도 되나요?**

A. 오늘 시나리오는 연습용입니다. 실제 업무에서는 실제 상황에 맞는 공개 데이터를 다시 가져오세요. 프롬프트 구조는 그대로 쓰시되 데이터를 바꾸시면 됩니다.
