### [S-12] 절대 금지 ④ AI는 자신 있게 틀립니다

**네 번째 위험: AI 환각(Hallucination)**

AI가 그럴듯하게 생성했지만 사실이 아닌 것들:

- 존재하지 않는 판례·법령 조항 인용
- 틀린 재무수치 ("영업이익 1.2조" → 실제 0.8조)
- 오래된 금리·시세 데이터
- 없는 사람·기업 이름

**→ 수치 / 날짜 / 법령 조항은 반드시 원출처 대조**

원출처: DART / 한국은행 / 금감원 / Bloomberg

★ 디자인: AI 말풍선에 오류 표시 + 원출처 아이콘

### [S-17] AI 출력 검증 3단계

**받았다고 바로 쓰면 안 됩니다**

**STEP 1. 수치·사실 확인**
숫자 / 날짜 / 법령 조항 → 원출처로 100% 대조

**STEP 2. 컴플 자가 체크**
개인정보 / MNPI / 미공개 정보가 출력에 섞여 있지 않은가?

**STEP 3. 상급자 리뷰 전 최종 감수**
외부 발송 / 내부 보고 전 최소 1인 검토

> 처음 3개월은 의식적으로. 3개월 후엔 자동으로 됩니다.

★ 디자인: 3단계 계단형 또는 순서도

### [S-18] AI는 어떻게 작동하나

**"다음 단어를 예측하는 기계"**

AI가 하는 일:
1. 지금까지 쓴 텍스트를 분석
2. 다음에 올 단어의 확률을 계산
3. 가장 그럴듯한 단어를 선택해 이어 붙임

"오늘 코스피는 ___" → [하락했습니다 42%] [상승했습니다 38%] [보합권 20%]

> 구글 검색과 다릅니다.
> 검색은 "찾는다" → AI는 "생성한다"
> 그래서 없는 것도 그럴듯하게 만들어냅니다.

★ 디자인: 텍스트 완성 애니메이션 or 확률 바차트

### [S-19] AI의 3가지 한계

**알고 쓰면 두렵지 않습니다**

🌀 **환각 (Hallucination)**
모르면 그냥 만들어냄. 수치·법령 오류 주의.

📅 **지식 컷오프**
학습 데이터에 기한이 있음. 최신 공시·금리는 모름.

📏 **맥락 창 제한**
한 번에 처리할 수 있는 텍스트 양에 한계 있음.
긴 IM·계약서를 통째로 넣으면 뒷부분을 놓칠 수 있음.

> **이 한계가 검증 3단계가 필요한 이유입니다.**

### [S-20] 좋은 프롬프트 = 4요소

**역할 + 맥락 + 지시 + 형식**

👤 **역할 (Persona)** — "당신은 WM 부서 시니어 어드바이저입니다"

📖 **맥락 (Context)** — 공개 자료 원문, 배경 정보 붙여넣기

📝 **지시 (Task)** — 무엇을, 어떤 목적으로, 누구에게

📐 **형식 (Format)** — 표 / 불릿 / 몇 자 이내 / 어떤 톤으로

> 4요소를 갖춘 프롬프트는 결과물이 **바로 쓸 수 있는 수준**으로 나옵니다.

★ 디자인: 4분할 카드, 각 요소 다른 컬러 포인트

---

### [S-21] 프롬프트 작성 예시 (WM)

```
[역할] 당신은 하나증권 WM 부서 어드바이저입니다.

주의사항 안내 메일 초안을 작성해 주세요.

[형식] 200자 이내, 전문용어 최소화, 문장체로
```

> 고객 이름·계좌번호는 절대 포함하지 마세요 🔴

### [S-23] 프롬프팅 팁 & 흔한 실수

**바로 쓸 수 있는 팁 5가지**

✅ 역할을 항상 먼저 설정하기
✅ 공개 원문을 직접 붙여넣기 (요약 말고)
✅ 출력 길이·형식 명시하기
✅ "수치는 추정하지 마세요" 명시하기
✅ 단계별로 나눠서 요청하기

**흔한 실수 3가지**

❌ 너무 짧고 막연한 지시
❌ 데이터 없이 수치 생성 요청
❌ 검증 없이 바로 붙여넣기

### [S-부록1] 공통 프롬프트 템플릿 4가지

**어디서나 쓸 수 있는 기본 틀**

**① 번역**
```
다음 영문을 한국어로 번역해줘.
금융 용어는 국내 증권업계 표준으로, 영문 원어 괄호 병기.
[텍스트]
```

**② 요약**
```
다음 문서의 핵심을 요약해줘.
핵심 메시지 1~2문장 + 주요 포인트 3~5개 + 핵심 수치.
출처에 없는 내용은 추가하지 마.
[문서]
```

**③ 교열**
```
다음 텍스트의 맞춤법·비문·어색한 표현을 교정해줘.
변경 부분을 [원문 → 수정] 형식으로 표시해줘. 내용은 바꾸지 마.
[텍스트]
```

**④ 표 정리**
```
다음 데이터를 마크다운 표로 정리해줘.
컬럼: [항목 나열]. 없는 값은 "-"로.
[데이터]
```
