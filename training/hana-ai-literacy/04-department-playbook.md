# 부서별 AI 활용 플레이북
## 하나증권 신입사원 AI 리터러시 교육 — M4 모듈 부서별 참고 자료

> **읽는 법**: 각 부서 섹션은 ① 신입이 자주 맡는 업무 → ② AI 활용 방법 → ③ 프롬프트 예시 → ④ Do/Don't 체크리스트 순으로 구성됩니다.
>
> 신호등 기호: 🔴 절대 금지 | 🟡 사내 승인 도구만 | 🟢 공용 AI 가능

## 1. WM / PB (자산관리·프라이빗뱅킹)

### 신입이 자주 맡는 업무
1. 일일/주간 시황 요약 자료 작성 (팀 공유 또는 고객 발송용)
2. 고객 문의·상담 후속 메일 작성
3. 상품 비교표 작성 (펀드, ETF, ELS 등)
4. 세미나·자료 준비 (PPT 초안, 시장 자료 정리)

### AI로 단축할 수 있는 것

| 업무 | 기존 소요 시간 | AI 활용 후 | 단축률 |
|------|--------------|-----------|--------|
| 일일 시황 요약 (공시·뉴스 기반) | 40~60분 | 10~15분 | ~70% |
| 고객 안내 메일 초안 | 20~30분 | 5~10분 | ~65% |
| 상품 비교표 초안 | 60~90분 | 20~30분 | ~65% |
| 세미나 자료 텍스트 초안 | 2~3시간 | 40~60분 | ~60% |

### 프롬프트 예시

**예시 1: 일일 시황 요약**
```
[역할] 당신은 하나증권 WM 부서에서 고객 담당 어드바이저 역할을 합니다.
[맥락] 오늘(2026년 4월 X일) 주요 시장 뉴스:
- 코스피 2,650 (-0.8%), 코스닥 870 (-1.2%)
- 미국 연준 의사록 공개: 금리 동결 기조 유지
- 유가 배럴당 75달러 (+2%)
[지시] 위 내용을 바탕으로 고객에게 보낼 오늘의 시황 요약을 작성해 주세요.
[형식] 3~4문장, 전문 용어 최소화, 친근하고 전문적인 톤, 수치 1~2개만 포함
```

**예시 2: 고객 후속 메일**
```
[역할] WM 부서 신입 어드바이저
[맥락] 오늘 50대 고객(가상: A고객)이 채권 금리 전망에 대해 문의했고, 
한국은행 기준금리 인하 사이클 관련 설명을 드렸습니다.
[지시] 오늘 상담 내용 감사 인사와 함께, 채권형 상품 관심 고객에게 
다음 단계(상품 제안 미팅 제안)를 자연스럽게 안내하는 후속 메일 초안을 써주세요.
[형식] 200자 이내, 정중하고 친근한 톤, 고객 실명·계좌번호 절대 사용 금지
```

**예시 3: 상품 비교표**
```
[역할] WM 상품 분석 보조
[맥락] 공개된 다음 3개 채권형 ETF 정보를 비교합니다:
- KODEX 국고채10년: 운용사 삼성자산, 보수 0.05%, 최근 1년 수익률 3.2%
- TIGER 국채3년: 운용사 미래에셋, 보수 0.07%, 최근 1년 수익률 2.1%
- KBSTAR 단기통안채: 운용사 KB자산, 보수 0.03%, 최근 1년 수익률 1.9%
[지시] 위 3개 ETF를 비교하는 표를 만들어줘. 
항목: 운용사, 보수, 1년 수익률, 만기 특성, 적합한 투자자 유형.
[형식] 마크다운 표, 마지막 행에 한 줄 요약 코멘트 추가
```

### WM 부서 Do/Don't

| ✅ Do (해도 됨) | 🚫 Don't (하면 안 됨) |
|---------------|---------------------|
| 공개 뉴스·한은 보도자료 기반 시황 요약 🟢 | 고객 실명·계좌번호 입력 🔴 |
| 공식 배포 상품 브로셔 기반 FAQ 작성 🟢 | 고객 포트폴리오 잔고·수익률 입력 🔴 |
| 가상 고객 프로필로 상품 적합성 검토 🟢 | CRM 데이터 다운로드 후 AI 입력 🟡 |
| 공개 지수·금리 데이터 분석 요청 🟢 | 미공개 금리 전망 보고서 입력 🔴 |
| 메일 초안 작성 (고객 정보 없이) 🟢 | 고객 상담 메모 그대로 입력 🔴 |

## 2. IB (기업금융·투자은행)

### 신입이 자주 맡는 업무
1. 산업 리서치 자료 수집·정리 (영문 포함)
2. IM(투자설명서) 특정 섹션 초안 작성
3. 비교회사 분석(Comps) 표 정리
4. 실사(DD) 자료 정리 및 요약
5. 영문 계약서·자료 번역

### AI로 단축할 수 있는 것

| 업무 | 기존 소요 시간 | AI 활용 후 | 단축률 |
|------|--------------|-----------|--------|
| 영문 산업 리포트 한국어 요약 | 2~3시간 | 30~45분 | ~75% |
| IM 섹션 목차·초안 | 3~4시간 | 1~1.5시간 | ~65% |
| Comps 표 구조 잡기 | 1~2시간 | 20~30분 | ~70% |
| 단순 영문 계약 조항 번역 | 1~2시간/장 | 15~20분 | ~75% |

### 프롬프트 예시

**예시 1: 산업 리서치 번역 요약**
```
[역할] IB 분석 보조, 국내 증권사 기업금융팀
[맥락] 아래는 Reuters에 오늘 공개된 글로벌 반도체 M&A 동향 기사입니다:
[기사 원문 붙여넣기 — 공개된 기사만]
[지시] 위 기사를 한국어로 번역하고, 다음 3가지를 구조화해서 정리해 주세요:
1. 핵심 딜 현황 (기업명, 규모, 거래 형태)
2. 시장이 주목하는 이유
3. 한국 반도체 기업에 주는 시사점
[형식] 불릿 포인트, A4 반 장 분량, 전문 용어는 영문 병기
```

**예시 2: IM 섹션 목차 구조화**
```
[역할] IB 팀의 IM 작성 보조 역할
[맥락] 가상 시나리오: 국내 중견 제조업체(연매출 3,000억)의 사모펀드 매각 딜.
실제 딜 정보, 코드명, 미공개 수치 없음.
[지시] 제조업 PE 딜 IM에서 "사업 현황 및 경쟁력" 섹션의 일반적인 목차 구조와
각 소섹션에서 다루어야 할 핵심 내용을 알려줘.
[형식] 목차 번호 포함, 각 소섹션별 2~3줄 설명
```

**예시 3: 비교회사 분석 표 구조**
```
[역할] 재무분석 보조
[맥락] 국내 상장된 2차전지 소재 기업 5개사의 EV/EBITDA, PER, 
영업이익률을 비교하는 Comps 표를 만들려고 합니다.
실제 수치는 Bloomberg·DART에서 직접 입력할 예정이고,
지금은 표의 구조와 형식만 잡아주세요.
[지시] EV/EBITDA, PER, EV/Sales, 영업이익률, 순부채비율 항목으로 
비교회사 분석 표 템플릿을 만들어줘. 
2024·2025·2026E 연도 컬럼 포함.
[형식] 마크다운 표
```

### IB 부서 Do/Don't

| ✅ Do (해도 됨) | 🚫 Don't (하면 안 됨) |
|---------------|---------------------|
| 공개 외신·공시 기반 산업 리서치 번역·요약 🟢 | 딜 코드명·대상 기업명 (미공개 시) 🔴 |
| 가상 딜 시나리오로 IM 목차·구조 연습 🟢 | 미공개 재무제표·밸류에이션 수치 🔴 |
| Comps 표 구조·템플릿 생성 (수치 없이) 🟢 | 실사 자료·NDA 보호 문서 🔴 |
| 공개 계약 조항 번역 (표준 조항) 🟢 | 서명 전 실제 계약서 전문 🔴 |
| Bloomberg 공개 데이터 기반 분석 요청 🟢 | 딜 참여 당사자 정보·지분 구조 (미공개) 🔴 |

## 3. S&T (세일즈&트레이딩)

### 신입이 자주 맡는 업무
1. 영문 글로벌 뉴스 빠른 번역·요약 (장중 실시간)
2. 매크로 경제 지표 캘린더 정리
3. 트레이딩 관련 엑셀/파이썬 코드 보조
4. 상품·전략 설명 자료 초안 작성

### AI로 단축할 수 있는 것

| 업무 | 기존 소요 시간 | AI 활용 후 | 단축률 |
|------|--------------|-----------|--------|
| 영문 뉴스 요약 (장중 5건) | 60~90분 | 15~20분 | ~75% |
| 매크로 캘린더 요약 정리 | 30~45분 | 10분 | ~70% |
| 엑셀 함수/VBA 코드 작성 | 30~60분 | 5~15분 | ~75% |
| 파이썬 데이터 처리 코드 | 1~2시간 | 20~30분 | ~70% |

### 프롬프트 예시

**예시 1: 장중 뉴스 요약**
```
[역할] S&T 데일리 브리핑 작성 보조
[맥락] 다음은 오늘 오전 공개된 주요 영문 뉴스 헤드라인·요약입니다:
1. Fed minutes show concern over inflation persistence (Bloomberg)
2. China manufacturing PMI fell to 49.2 in March (Reuters)
3. TSMC Q1 revenue beats estimates, raises 2026 outlook (FT)
[지시] 위 3개 뉴스를 한국어로 요약하고, 각각이 국내 주식시장(코스피)에 주는 
단기 영향을 한 줄씩 덧붙여 주세요.
[형식] 번호 목록, 뉴스 요약 2~3줄 + 시장 영향 1줄, 전문 용어 영문 병기
```

**예시 2: 엑셀 함수 보조**
```
[역할] 엑셀 전문가
[지시] 다음 상황에 맞는 엑셀 함수를 알려줘:
- A열에 날짜, B열에 종가 데이터가 있음 (A1부터 A250까지)
- C열에 20일 이동평균을 계산하고 싶음
- 처음 19개 셀은 N/A로 표시
실제 데이터는 없고 수식 로직만 필요.
[형식] 수식 예시 + 각 인수 설명
```

**예시 3: 파이썬 백테스트 코드 구조**
```
[역할] 파이썬 금융 데이터 분석 보조
[지시] pandas를 사용해서 일별 수익률을 계산하고 
샤프비율(Sharpe Ratio)을 구하는 함수를 작성해 줘.
입력: 일별 종가 시리즈 (pandas Series), 무위험 수익률(연율화)
출력: 연율화 샤프비율 (float)
실제 시세 데이터 없이 함수 로직만.
[형식] 파이썬 코드 + 주석 + 사용 예시
```

### S&T 부서 Do/Don't

| ✅ Do (해도 됨) | 🚫 Don't (하면 안 됨) |
|---------------|---------------------|
| 공개 뉴스·외신 번역 요약 🟢 | 실시간 호가·주문 데이터 🔴 |
| 공개 PMI·금리 등 매크로 지표 정리 🟢 | 자사 트레이딩 포지션·리스크 노출 🔴 |
| 엑셀/파이썬 코드 (실데이터 없이) 🟢 | 고객 거래 내역·계좌 정보 🔴 |
| 공개 시장 데이터 기반 전략 아이디어 토론 🟢 | 내부 트레이딩 전략·모델 파라미터 🟡 |
| 금융 용어·상품 구조 설명 요청 🟢 | Bloomberg 터미널 민감 데이터 스크린샷 🔴 |

## 4. 리서치 (기업분석·채권분석)

### 신입이 자주 맡는 업무
1. DART 사업보고서·공시 수치 정리
2. 글로벌 리서치 리포트 번역·요약
3. 차트·그래프 캡션 초안 작성
4. 리포트 편집·교열 보조

### AI로 단축할 수 있는 것

| 업무 | 기존 소요 시간 | AI 활용 후 | 단축률 |
|------|--------------|-----------|--------|
| DART 공시 수치 표 정리 | 60~90분 | 15~20분 | ~75% |
| 영문 리포트 번역 요약 | 2~3시간 | 30~45분 | ~75% |
| 차트 캡션 초안 | 15~20분/개 | 3~5분/개 | ~75% |
| 리포트 섹션 교열 | 30~60분 | 10~15분 | ~70% |

### 프롬프트 예시

**예시 1: DART 공시 수치 정리**
```
[역할] 기업 재무분석 보조
[맥락] 다음은 DART에서 가져온 A기업 2025년 사업보고서 발췌문입니다:
[공시 텍스트 붙여넣기]
[지시] 위 내용에서 다음 항목을 표로 정리해 주세요:
- 매출액, 영업이익, 당기순이익 (2023, 2024, 2025 3개년)
- 영업이익률, 당기순이익률 (소수점 1자리)
- 부채비율, 유동비율
[형식] 마크다운 표, 단위는 억원, 비율은 % 표기
주의: 수치는 원문 그대로 옮겨줘. 절대 추정하지 마.
```

**예시 2: 글로벌 리포트 번역 요약**
```
[역할] 리서치팀 번역·요약 보조
[맥락] 아래는 Goldman Sachs에서 공개 배포한 반도체 섹터 리포트 일부입니다:
[원문 붙여넣기]
[지시] 위 내용을 한국어로 번역하고, 다음 구조로 요약해 주세요:
1. 투자 포인트 (3개)
2. 리스크 요인 (2개)
3. 주요 수치·추정치 표
[형식] 전문 용어 영문 병기, 수치는 원문 그대로 사용
주의: 타사 리포트를 자사 의견인 것처럼 표현하지 말 것.
```

**예시 3: 차트 캡션 작성**
```
[역할] 리서치 리포트 편집 보조
[맥락] 다음 차트를 설명하는 캡션을 작성하려 합니다:
- 차트 종류: 막대그래프
- X축: 2020~2025년 (연도별)
- Y축: 국내 2차전지 수출액 (억 달러)
- 트렌드: 2020년 50억달러 → 2025년 180억달러로 꾸준히 증가
[지시] 이 차트의 캡션(제목 + 1줄 설명)과 본문 인용 문장 1개를 작성해 주세요.
[형식] 캡션은 20자 이내, 인용 문장은 2~3줄
```

### 리서치 부서 Do/Don't

| ✅ Do (해도 됨) | 🚫 Don't (하면 안 됨) |
|---------------|---------------------|
| 공시(DART)·거래소 공개 자료 요약 🟢 | 발간 전 자사 리포트·목표주가 🔴 |
| 타사 공개 배포 리포트 번역·요약 🟢 | 내부 애널리스트 의견·초안 🟡 |
| 차트 캡션·섹션 교열 보조 🟢 | 미발표 EPS 추정치·컨센서스 이탈 정보 🔴 |
| 공개 재무 데이터 기반 분석 요청 🟢 | 기업 IR 미팅 중 입수한 미공개 정보 🔴 |
| 금융·산업 용어 번역 검토 🟢 | 리포트 발간 전 해당 종목 관련 일체 🔴 |

## 5. 리테일 / 지점 영업

### 신입이 자주 맡는 업무
1. 고객 투자 안내문·공지 작성
2. 투자 설명회·세미나 자료 보조
3. 상품별 FAQ·Q&A 자료 준비
4. 고객 민원·문의 답변 초안

### AI로 단축할 수 있는 것

| 업무 | 기존 소요 시간 | AI 활용 후 | 단축률 |
|------|--------------|-----------|--------|
| 고객 안내문 초안 | 60~90분 | 15~20분 | ~75% |
| 상품 FAQ 10문항 | 60~90분 | 15~25분 | ~70% |
| 세미나 자료 텍스트 초안 | 2~3시간 | 40~60분 | ~65% |
| 민원 답변 초안 | 30~60분 | 10~15분 | ~70% |

### 프롬프트 예시

**예시 1: 고객 투자 안내문**
```
[역할] 리테일 영업 지원 담당
[맥락] 하나증권이 새로운 공모 채권형 펀드(가상 상품명: 하나안심채권펀드1호)를 출시했습니다.
상품 특징: 국내 국공채 중심, 목표 수익률 연 4.5%, 최소 가입 100만원, 
환매 가능(익일 기준), 운용 기간 1년.
(위는 교육용 가상 상품입니다)
[지시] 이 상품을 처음 접하는 60대 고객에게 보낼 투자 안내 메일 초안을 작성해 주세요.
[형식] 150자 이내, 전문 용어 쉬운 말로 풀어서, 투자 원금 손실 가능성 문구 포함
주의: 수익률 보장 표현 절대 금지. 금소법 준수.
```

**예시 2: 상품 FAQ**
```
[역할] 고객 응대 지원 담당
[맥락] 하나증권의 ELS 상품(가상)에 대해 고객들이 자주 하는 질문을 정리합니다.
ELS 기본 구조: 기초자산 연계, 조건 충족 시 조기 상환, 원금 손실 가능.
[지시] ELS 상품에 관한 고객 FAQ 10개를 Q&A 형식으로 만들어 주세요.
가능한 한 고객 관점에서 쉬운 말로.
[형식] Q: 질문 / A: 2~3줄 답변, 위험 관련 질문은 반드시 원금 손실 가능성 명시
주의: 수익 보장·원금 보장 표현 절대 금지.
```

**예시 3: 고객 민원 답변 초안**
```
[역할] 고객 민원 처리 담당
[맥락] 고객(비식별 처리: "50대 고객")이 주식 매매 체결 지연에 대해 
불만 접수를 했습니다. 시스템 점검으로 인한 5분 지연이었으며 
손실은 발생하지 않았습니다.
[지시] 이 고객에게 보낼 공식 답변 초안을 작성해 주세요.
[형식] 사과 → 원인 설명 → 재발 방지 약속 → 고객 신뢰 회복 문구 순서로,
200자 이내, 정중하고 공감적인 톤
주의: 고객 실명·계좌 없이 작성. 법적 책임 인정 표현 금지.
```

### 리테일/지점 Do/Don't

| ✅ Do (해도 됨) | 🚫 Don't (하면 안 됨) |
|---------------|---------------------|
| 공식 배포 상품 자료 기반 안내문 초안 🟢 | 고객 실명·계좌·거래내역 입력 🔴 |
| 가상 고객 프로필로 응대 시뮬레이션 🟢 | 고객 상담 메모·CRM 내용 입력 🔴 |
| 공개 상품 정보 기반 FAQ 작성 🟢 | 고객 포트폴리오 현황 입력 🔴 |
| 민원 답변 초안 (비식별화) 🟢 | 미공개 신상품 스펙·가격 🔴 |
| 금융 개념 쉬운 말 풀이 요청 🟢 | 고객 투자 성향 원본 데이터 🔴 |

## 6. 백오피스 / IT / 컴플라이언스

### 신입이 자주 맡는 업무
1. 엑셀 데이터 정리·함수 작성
2. 회의록·보고서 초안 정리
3. 내부 규정·지침 검색·요약 (공개 법령 기준)
4. 파이썬/VBA 코드 작성 보조
5. 업무 프로세스 문서화

### AI로 단축할 수 있는 것

| 업무 | 기존 소요 시간 | AI 활용 후 | 단축률 |
|------|--------------|-----------|--------|
| 엑셀 복잡 함수 작성 | 30~60분 | 5~10분 | ~80% |
| VBA 매크로 초안 | 2~3시간 | 30~45분 | ~75% |
| 회의록 초안 (녹취 기반) | 45~60분 | 10~15분 | ~75% |
| 공개 법령 조항 요약 | 60~90분 | 10~15분 | ~80% |

### 프롬프트 예시

**예시 1: 엑셀 복잡 함수**
```
[역할] 엑셀 전문가
[지시] 다음 요구사항에 맞는 엑셀 함수를 작성해 줘:
- B열: 부서명 (예: WM, IB, ST, 리서치)
- C열: 직급 (예: 신입, 대리, 과장)
- D열: 월급여 (숫자)
- E열에: B=WM이고 C=신입인 경우에만 D값을 합산하는 함수
- 조건이 맞지 않으면 빈칸
실제 데이터 없이 수식만.
[형식] =수식 + 각 인수 설명 1줄씩
```

**예시 2: VBA 매크로**
```
[역할] VBA 프로그래밍 보조
[지시] 다음 기능을 하는 엑셀 VBA 매크로를 작성해 줘:
1. 현재 시트에서 A열 기준으로 중복값 행 삭제
2. 처리 후 "중복 X건 삭제 완료" 메시지 박스 표시
3. 실행 전 백업을 위해 현재 시트를 새 시트로 복사
실제 데이터 없이 로직만.
[형식] VBA 코드 + 각 Sub 설명 주석
```

**예시 3: 공개 법령 요약**
```
[역할] 컴플라이언스 법령 분석 보조
[맥락] 금융감독원 공식 사이트에 공개된 금융소비자보호법 제19조(설명의무) 
조항 텍스트:
[조항 원문 붙여넣기]
[지시] 위 조항을 신입 직원이 이해할 수 있도록 쉬운 말로 요약해 줘.
포함 항목: 의무 주체, 설명해야 할 사항, 위반 시 결과
[형식] 3개 항목 불릿 포인트, 200자 이내
주의: 법령 해석은 참고용이며 실제 적용은 컴플팀 확인 필수라고 명시해 줘.
```

**예시 4: 회의록 초안 (비식별화 후)**
```
[역할] 회의록 작성 보조
[맥락] 다음은 오늘 팀 주간 회의 내용을 텍스트로 정리한 것입니다.
실명 대신 A팀장, B대리 등으로 표기했습니다:
[회의 내용 텍스트 붙여넣기]
[지시] 위 내용을 정식 회의록 형식으로 정리해 줘.
포함 항목: 회의 일시, 참석자(비식별), 안건, 주요 논의 사항, 결정 사항, 후속 조치
[형식] 표 형식, 결정 사항과 후속 조치는 담당자 코드·완료 기한 포함
```

### 백오피스/IT/컴플 Do/Don't

| ✅ Do (해도 됨) | 🚫 Don't (하면 안 됨) |
|---------------|---------------------|
| 엑셀 함수·VBA 코드 (실데이터 없이) 🟢 | 인사 자료·급여 데이터 원본 🔴 |
| 파이썬 코드 로직 작성 (실데이터 없이) 🟢 | 내부 감사 보고서·징계 기록 🔴 |
| 공개 법령 텍스트 요약·풀이 🟢 | 사내 IT 시스템 취약점 정보 🔴 |
| 비식별화된 회의록 초안 정리 🟡 | 미서명 계약서·협약서 원문 🔴 |
| 업무 프로세스 문서화 초안 🟡 | 개인 성과평가 자료 🔴 |

## 공통: 부서 무관 바로 쓸 수 있는 프롬프트 템플릿

### 영문 → 한국어 번역 (공개 문서)
```
다음 영문 텍스트를 한국어로 번역해 줘.
금융 전문 용어는 국내 증권업계 표준 용어로 번역하고, 
영문 원어를 괄호 안에 병기해 줘.
[텍스트 붙여넣기]
```

### 긴 문서 요약 (공개 자료)
```
다음 문서를 읽고 핵심 내용을 요약해 줘.
요약 구조: 
1. 핵심 메시지 (1~2문장)
2. 주요 포인트 3~5개 (불릿 포인트)
3. 숫자로 된 핵심 지표 (있는 경우)
출처에 없는 내용은 절대 추가하지 마.
[문서 붙여넣기]
```

### 보고서·메일 교열
```
다음 텍스트를 교열해 줘. 
수정 항목: 맞춤법, 비문, 어색한 표현, 불필요한 반복
수정 후 변경한 부분을 [원문 → 수정]으로 표시해 줘.
내용 자체는 바꾸지 마. 
[텍스트 붙여넣기]
```

### 표 구조화
```
다음 비정형 데이터를 정리된 표 형식으로 만들어 줘.
컬럼: [항목명 나열]
마크다운 표 형식으로 출력해 줘.
출처에 없는 값은 "-"로 표기해 줘.
[데이터 붙여넣기]
```
